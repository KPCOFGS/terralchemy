Getting Started — Everything happens here in the browser

1. Upload Data
Go to the Sources tab and drag & drop your geo files (Shapefile, GeoJSON, GeoPackage, etc.). We'll register them automatically.
2. Create Models
Go to the Models tab and click + New Model. Pick a template (filter, spatial join, buffer, etc.) and customize the SQL.
3. Add Tests
Go to Run Output tab and use the Test Builder to add quality checks — geometry validity, bounds, row counts.
4. Run & Preview
Click Run Pipeline at the top, then go to the Map Preview tab to see your results on a map.
