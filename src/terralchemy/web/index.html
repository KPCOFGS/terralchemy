Quick Start

1. Add Data
Drop geo files (Shapefile, GeoJSON, GeoPackage, etc.) into your project's data/ folder, then define them in sources/*.yml
2. Write Models
Create SQL files in models/ using spatial functions like ST_Buffer, ST_Within, ST_Intersects. Use the Models tab to edit.
3. Run & Test
Click Run Pipeline to execute, then Run Tests to validate. Preview results on the Map tab.
